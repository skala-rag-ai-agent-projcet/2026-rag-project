## 3. 메트릭 해석 가이드

- **Faithfulness ≥ 0.8**: 응답이 검색 문서에 충실하게 근거함 (환각 최소)
- **Response Relevancy ≥ 0.8**: 질문에 대해 적절한 답변을 생성
- **Context Precision ≥ 0.7**: 검색된 문서 대부분이 실제로 관련 있음
- **Context Recall ≥ 0.7**: 정답에 필요한 정보 대부분을 검색으로 확보

## 4. RAG 파이프라인 구성

```
문서(4 PDF) → 청킹(1000자/200 overlap)
  → FAISS 인덱싱(bge-m3 임베딩)
  → Corrective RAG:
      1) Top-5 검색
      2) LLM 기반 관련성 평가 (binary yes/no)
      3) 관련 문서 0개 → 쿼리 리라이트 → 재검색
      4) 여전히 0개 → 웹 검색 fallback
  → 에이전트 프롬프트에 컨텍스트 주입
```
